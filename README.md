## Features
- Username must contain "_" and be max 5 characters
- Password must be at least 8 characters with:
  - Capital letter
  - Number
  - Special character
- Phone number must start with +27

## How It Works
1. User enters username, password, and phone number
2. System validates all inputs
3. If correct → user is registered
4. User logs in using same details
5. System confirms success or failure

## Files
- Main.java → controls program flow
- Login.java → handles validation and login logic

## How to Run
1. Open project in NetBeans
2. Run Main.java
3. Follow console instructions
